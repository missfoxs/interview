6. 路由拦截和接口拦截

   路由拦截在beforeRoute方法中判断，先在路由中添加meta信息，然后根据meta信息判断没有登录时是否需要跳转到登录页面；

   接口拦截分两种，request拦截可以为每一个http请求添加头信息，response拦截作用到每一个http返回的response，根据状态码判断登录是否失效。

2. 使用v-if隐藏元素，发现元素只隐藏了一部分

   v-if是动态的向dom树内添加或者删除元素，v-show是控制display的属性。v-if是惰性的，初始条件为false的时候，会默认不加载。v-show是在任何条件下都被编译，然后被缓存，而且DOM元素保留；v-if有更高的切换消耗；v-show有更高的初始渲染消耗；v-if适合运营条件不大可能改变；v-show适合频繁切换。

8. element中事件和方法的区别，以及某些attr可以使用方法

   事件可以使用@进行监听，比如js中默认可以监听click事件，在element-ui中有一些自定义的监听事件，比如表格中有select-all， cell-click等事件。

   方法不可以使用@监听，但是可以通过表格调用，方式如下

   ```javascript
   this.$refs.table.clearSelection();
   ```

   组件中绑定的属性可以是方法，如下所示

   ```javascript
   <h1 :class="setColor('red')">测试</h1>
   setColor(color) {
       return {
           'color': color
       } 
   }
   ```
